# Get Convergence of bboufit Object

Get Convergence of bboufit Object

## Usage

``` r
# S3 method for class 'bboufit'
converged(x, rhat = 1.05, ...)
```

## Arguments

- x:

  The object.

- rhat:

  A number greater than 1 of the maximum rhat value required for model
  convergence.

- ...:

  Unused parameters.

## Value

A flag indicating convergence.
