# Get Model Code

Get code from Nimble model.

## Usage

``` r
model_code(x, ...)

# S3 method for class 'bboufit'
model_code(x, ...)

# S3 method for class 'bboufit_ml'
model_code(x, ...)
```

## Arguments

- x:

  The object.

- ...:

  Unused parameters.

## Methods (by class)

- `model_code(bboufit)`: Get model code from bboufit object.

- `model_code(bboufit_ml)`: Get model code from bboufit_ml object.
