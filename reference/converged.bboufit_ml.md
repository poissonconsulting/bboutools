# Get Convergence of bboufit_ml Object

Successful convergence indicates that no convergence warnings were
produced by optim and all standard errors could be estimated.

## Usage

``` r
# S3 method for class 'bboufit_ml'
converged(x, ...)
```

## Arguments

- x:

  The object.

- ...:

  Unused parameters.

## Value

A flag indicating convergence.
